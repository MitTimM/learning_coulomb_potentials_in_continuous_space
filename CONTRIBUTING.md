# Contributing

Thank you for your interest in contributing!

## Reporting Issues

Please open an issue on GitHub with:
- Clear description of the problem
- Steps to reproduce (if applicable)
- Python version and package versions (numpy, scipy, matplotlib)

## Code Contributions

1. Fork and create a branch: `git checkout -b feature/your-feature`
2. Make your changes and test them:
   ```bash
   python single_coulomb/single_main
   python multi_coulomb/main
   ```
3. Follow PEP 8 style and add docstrings for new functions
4. Push your branch and create a Pull Request

## Code Style

- Use clear variable names
- Add docstrings to functions and classes
- Keep lines under 100 characters
- Follow PEP 8 conventions
